# Sales-and-Profit-report-E-commerce

# E-Commerce Growth & Customer Insights Dashboard (Tableau)

## 🛒 Project Overview
This project analyzes a multi-channel E-commerce dataset to visualize sales performance, customer behavior, and ROI. The dashboard helps stakeholders understand which products drive the most revenue and which customer segments are the most loyal.

## 🔗 Live Dashboard
[Link to your Dashboard on Tableau Public]  https://github.com/SagarBegins/Sales-and-Profit-report-E-commerce/blob/main/Sales%20and%20profit%20dashboard%20for%20e-commerce.png

## 🎯 Key Business Questions Addressed
- **Revenue Growth:** What are the month-over-month sales trends?
- **Customer Segmentation:** Who are our "Power Users" vs. "At-Risk" customers?
- **Product Performance:** Which categories have the highest return rates?
- **Marketing Efficiency:** Which acquisition channels provide the highest Profit Margin?

## 🛠️ Data Processing & Tools
- **Tableau Desktop:** Used for data visualization, calculated fields, and dashboard interactivity.
- **Data Cleaning:** Performed data joins between Orders, Customers, and Marketing_Spend tables to create a unified view.
- **Calculated Fields:** Created custom metrics including **Customer Lifetime Value (CLV)**, **Retention Rate**, and **AOV (Average Order Value)**.

## 📊 Dashboard Highlights
- **Executive Summary:** High-level KPIs (Total Sales, Total Profit, Profit Margin).
- **Geographic Map:** Sales distribution by region to identify untapped markets.
- **Product Tree Map:** Visualizing category performance and stock levels.
- **Customer Retention Cohorts:** Analyzing how many customers return month-after-month.
